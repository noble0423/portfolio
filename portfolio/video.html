Bamazon App

(Node.js, MySQL)

Bamazon was created as part of a 2-day, individual homework assignment that required me to build an Amazon-like back-end storefront that could take in customer orders and update stock within the store's inventory.

At this point of the Coding Boot Camp, we were learning programming techniques that focused almost strictly on the back-end. This application uses Node.js, the Inquirer NPM module, and the Sequelize NPM module to communicate with an inventory database that was built in MySQL.

Coding this application taught me quite a bit about the varying ways that one can interact with a database. Furthermore, it displayed the need to be precise with your code because a simple mispunctuation can create database errors that can have detrimental effects to the client using the software.
